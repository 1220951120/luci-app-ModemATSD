# luci-app-zmodem-AK68

## 2.9-r20

- 手机端收件箱改为卡片布局，短信正文独占整行，避免被其他列挤压后每一两个字换行。

## 2.9-r19

- 修复 `10010`、`10086` 等国内短服务号码被错误编码为国际号码的问题。
- 发送结果改为“已提交给模组”，避免把模组返回 `OK` 误解为对端已经收到短信。

## 2.9-r18

- 将收件箱、发送短信、发送记录和 PushPlus 短信转发合并到“设备短信”页面。
- 发送短信改用独立 POST 接口，不再触发 CBI 的“保存设置”提示。
- 修复窄操作列导致“删除”按钮换行，并限制内存发送记录为最近 100 条。
- 转发状态接口不再返回完整 Token，更新转发设置时仅重启转发进程。

## 2.9-r17

- 使用 Python PDU 解码器替换旧的闭源解码程序，正确解析 UCS2、GSM 7-bit 和短信时间。
- 自动合并长短信分段，不再显示 `Reference number`、`SMS segment` 或错误的 `x0A` 发件人。
- 修复短信转发偶发空内容、空发件人和异常时间的问题。
- 短信删除改为受限的 POST 操作，不再执行页面传入的 shell 命令。
